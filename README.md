# README

# サービス概要
食生活が乱れがちな、健康を意識してきた方へ、健康レシピに特化した料理レシピ共有サービスです

# 登場人物
* エンドユーザー
    * 食生活が乱れてきていて健康や節約のために自炊をしようと決意したが何をどう作ればいいのかわからない方
* 管理者
    * ユーザーを管理する人
